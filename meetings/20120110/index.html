Agenda

Swisstrains.ch is a simulation of the Swiss Federal Railways network. The simulation is based on the SBB timetables and railroad vector data (track lines and stations) of Switzerland. The project remained frozen for almost 4 years but recently a new version kicked off with faster, better UX and enriched features.

The figures behind the project are quite impressive: the railroad network has more than 5,000 km in length, inside which run daily ca. 13,000 trains that generate 150,000 timetables stops across 1,800 stations. An open source vehicle simulator powers the new version, of which you'll get a short demo, will see how to use it in and how to to create your own public transport simulation. You'll hear all about the deeper motivation behind the project, the challenges encountered, the technology stack used and where it all goes next.

Speaker

Vasile Cotovanu is a mobile software engineer employed by local.ch ag; he is a map addict, loves hiking but also hacking and last but not least he is a train buff :)
