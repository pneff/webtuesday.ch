HipHop Virtual Machine (HHVM) for PHP

HHVM is a new open-source virtual machine designed for executing programs written in PHP. It is developed by Facebook and is much faster than a webserver with stock PHP. It also adds some new language features like type checking, generics, async operations and collections called Hack

This talk will go into the details of HHVM, show some examples of Hack and also tell about real life experience with it.

About Christian Stocker @chregu

Chregu is one of the partner and founders of LIIP AG. He's involved in the PHP community and was one of the main contributers for the XML and XSLT extensions in PHP. He and the team at Liip he's working with is currently using HHVM in a big project for Migros.
