Agenda

  • Topic: XHProf - facebooks PHP profiler
  • Speaker: Harry Fuecks

XHProf, the PHP profiler written by facebook to allow performance analysis of live web applications. You may have used xdebug already and one thing I'll be discussing is why you want both. We've been using XHProf at local.ch since February 2010 so there's one or two "war stories" to be told as well. Hopefully an interesting talk for PHP geeks or those having to deal with higher traffic web sites.
