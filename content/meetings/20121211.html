Agenda

Sebastian will give a talk about Semantic Web aka Web3.0.

Topics

  1. Quick walk through the history of the Web
  2. The core principles of the Semantic Web:
    • resolvable identifies (URIs)
    • RDF - distributed, directed graph of interlinked resources
    • vocabularies and ontologies
  3. Semantic Web gotchas:
    • open world assumptions
    • inheritance
  4. The vocabularies, schemas and ontologies everyone should know about:
    • Dublin Core
    • FOAF, SIOC, SKOS
    • Good Relations
    • schema.org
  5. Publishing the Web 3.0 way:
    • Linked Open Data
    • content negotiation
    • RDFa
    • rich snippets
    • Good Relations
  6. Consuming semantics:
    • quick overview of SPARQL
  7. Summary: how Semantic Web technologies can help
    • interoperability
    • information retrieval
    • recommendations and matchmaking
    • SEO

Speaker

Sebastian Kruk (@skruk) works at Mila as a Domain Expert Web3.0
